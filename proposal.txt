Project Overview:
My project would simulate a single ecosystem made up 
of predators, prey, and producers laid out as a 2d grid
of cell locations. Each 'tic' of discrete time would cause
the producers/plants to grow, herbivores to consume and reproduce, and 
predators to hunt and reproduce. Population counts would be tracked
per cell and simple migration between cells would be possible. 
The program would run with certain initial conditions and run for
at least 25 time periods.
